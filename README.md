## Setup

**Install dependencies**

> npm i

**Build the project**

> npm run build

## Run in local

1. Run `npm run build` and this will generate a `dist` folder
1. Navigate to **chrome://extensions/**
1. Click on *developer mode*
1. Click on *Load unpacked* and upload the `dist` folder
